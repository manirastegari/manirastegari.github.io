Explore City: Tourism App

Category: iOS / Swift

The project involves developing an application that allows users to discover nearby events such as concerts or sports games, and connect with other users interested in similar events. This app builds upon a previous course project and integrates features with the SeatGeek API for event data and offers options for data persistence through Firetore Database services.

Here are the main features implemented:

  • Sign-up, login, and logout functionalities for users.
  • Display a list of events of the city the user searches.
  • Implement a screen with a map displaying clickable markers or a list format.
  • Users can select an event from the list to view more details.
  • Users can indicate their attendance, adding the event to their list.
  • A screen where users can view their friends list.

Technologies used: Xcode

Project Duration: 2 weeks

Overview: This project involves the development of a mobile application aimed at enhancing the event discovery experience for users, enabling them to find nearby events and connect with others who share their interests. The application gets the data from the SeatGeek API and stores information to the database.
