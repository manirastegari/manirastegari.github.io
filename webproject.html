Car Rental App

Category: Cross-platform / React Native

Car Owner

Renter

This project entails the creation of two React Native applications for an electric car rental service: the Car Owner app and the Renter app. The Car Owner app enables users to post their electric vehicles for rent and manage bookings, while the Renter app allows users to search for available vehicles and make bookings.

Here are the main features implemented:

  • Users can create rental listings for their electric vehicles using a form.
  • Users can prepopulate the form with data from the API and modify any values.
  • Data is saved to the database, including geocoded location for querying nearby listings.
  • Car owners can view and manage their rental bookings.
  • Owners can approve or decline booking requests.
  • Renter app displays a map of all rental listings in the user's city based on device location.
  • Custom map markers represent each listing with the price displayed. Clicking a marker shows a summary of the listing
  • The "BOOK NOW" button attempts to book the vehicle for a random future date.
  • My Reservations Screen shows a list of the user's rental bookings.

Technologies used: Visual Studio Code

Project Duration: 2 weeks

Overview: This project involves the development of two applications designed for a car rental service: the Car Owner app and the Renter app. The goal is to provide a platform where electric car owners can list their vehicles for rent, manage bookings, and where renters can search for available vehicles, make bookings, and view their reservations.
