4Rent.ca - Rental App

Category: Android / Kotlin

This project involves the development of an Android application for managing rental properties, catering to three types of users: Guest Users, Tenant Users, and Landlord Users. The app includes features such as property listing, searching, shortlisting, and property posting functionalities.

Here are the main features implemented:

  • The first screen allows users to enter a city name, property address, or postal code to search for available rentals.
  • Search results are displayed using a RecyclerView showing matching rental properties.
  • Tenant users can search for rentals like guest users but when they select a rental from the search results, they are prompted to log in or create an account.
  • The selected rental is then added to the tenant's shortlisted rentals.
  • Tenants can remove items from their shortlist, and selecting a property from the list opens a details screen with address, rent, description, etc.
  • Landlord users can post their properties for rent.
  • Upon selecting this option, landlords are prompted to log in or create an account. Once logged in, they can provide details for their properties.

Technologies used: Android Studio Code

Project Duration: 2 weeks

Overview: This project aims to create a user-friendly application for managing rental properties, offering a seamless experience for guests to search, tenants to shortlist, and landlords to post and manage their rental properties. The use of SharedPreferences ensures data persistence across user sessions, enhancing the usability and convenience of the application.
